## 算法
今天做的算法题是：字符串中的第一个唯一字符
解题思路：第一次遍历用HashMap统计各字符出现的次数，第二次遍历找出只出现一次的字符，当找到第一个只出现一次的字符就返回下标。
时间复杂度：O(n)
空间复杂度：O(k)，k 为字符种类

## Java 四大特性是什么？
封装：私有属性和方法，只开放关键方法，保护类属性；
继承：一个父类可以有多个子类，子类可访问父类的属性和方法，还可以定义自己的属性和方法；
多态：父类或接口引用可以指向子类或实现类对象，调用被重写的方法时，运行时会执行实际对象对应的实现。；
抽象：抽象的本质是提取共性、隐藏细节；抽象类可以有字段、构造器和已实现的方法，接口也可以有 default、static 方法。
## 重载和重写的区别？
重写是子类重写父类的方法，重载是指同一个类允许同一方法名，但参数类型、参数个数不一样。
## == 和 equals() 的区别？
==在基础类型比较值，引用类型比较是否指向同一个对象；
equals()默认仍是比较引用，重写后才按内容或业务字段比较。String 的 equals() 就是按字符内容比较。。
## String 为什么不可变？
1、String类被final修饰，意味着类不能被继承；
2、String类的value属性被final修饰，意味着数组引用不能换指向；
3、String类不对外开放value属性，也不提供修改方法，类本身也没有修改value的方法。
## 接口和抽象类怎么选？
接口主要定义能力和规范，可以定义常量，也可以提供默认实现；
抽象类除了声明方法，也可以实现方法，还可以有属性；
怎么选：需要定义能力、支持多实现时用接口；需要复用状态和公共逻辑时用抽象类。
