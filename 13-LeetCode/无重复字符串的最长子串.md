## 给定一个字符串 s ，请你找出其中不含有重复字符的 最长 子串 的长度。

思路是用一个滑动窗口表示当前没有重复字符的子串：[left, right]
right 从左到右遍历字符串。
lastIndexMap 记录每个字符最后一次出现的下标。
如果当前字符以前出现过，并且它还在当前窗口内，就把 left 移到它上一次出现位置的后一位。
每轮计算当前窗口长度：right - left + 1。
例如 "abba"：
right=0, a：窗口 "a"，长度 1
right=1, b：窗口 "ab"，长度 2
right=2, b：b 上次在 1，left 移到 2，窗口 "b"
right=3, a：a 上次在 0，但已经在 left 左侧，不移动 left，窗口 "ba"，长度 2

import java.util.HashMap;
import java.util.Map;

class Solution {
public int lengthOfLongestSubstring(String s) {
Map<Character, Integer> lastIndexMap = new HashMap<>();

        int left = 0;
        int maxLength = 0;

        for (int right = 0; right < s.length(); right++) {
            char currentChar = s.charAt(right);

            Integer lastIndex = lastIndexMap.get(currentChar);
            if (lastIndex != null) {
                left = Math.max(left, lastIndex + 1);
            }

            lastIndexMap.put(currentChar, right);
            maxLength = Math.max(maxLength, right - left + 1);
        }

        return maxLength;
    }
}
